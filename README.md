# Biology Lessons Landing Page

A responsive landing page for biology tutor website concept.

## Live Demo
https://makbridgecoder.github.io/biology-lessons-landing-page/

## Features
- Responsive layout using Flexbox and media queries
- Hero section with layered image
- Lesson cards section
- Newsletter form with basic validation
- Smooth scroll to the "Classes" section

## Build With 
- HTML5
- CSS3
- Google Fonts
- Font Awesome

## What I Practiced
- Workin with responsivenes design breakpoints
- Building reusable layout containers (row / column system)
- Improving form accessibility (labels + semantic structure)
- Using Git and GitHub workflow (branch + PR)
